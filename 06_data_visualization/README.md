# Data Visualization

This module covers creating insightful visualizations using Matplotlib and Seaborn.

## Learning Objectives

- Create basic plots (line, bar, scatter, histogram)
- Use Seaborn for statistical visualizations
- Customize plot appearance
- Create multi-plot figures
- Visualize distributions and relationships

## Examples Included

1. **matplotlib_basics.py**: Basic plotting with Matplotlib
2. **seaborn_visualizations.py**: Statistical plots with Seaborn
3. **advanced_visualizations.py**: Advanced visualization techniques

## Key Concepts

### Matplotlib
- Figure and axes
- Plot types (line, bar, scatter, pie, histogram)
- Customization (colors, labels, legends, styles)
- Subplots and layouts

### Seaborn
- Distribution plots
- Categorical plots
- Relationship plots
- Heatmaps and matrix plots
- Style and color palettes

## Running the Examples

```bash
cd 06_data_visualization
python matplotlib_basics.py
python seaborn_visualizations.py
python advanced_visualizations.py
```

# Text Classification

This module introduces text processing and classification techniques.

## Learning Objectives

- Text preprocessing techniques
- Feature extraction (TF-IDF, Count Vectorizer)
- Text classification using machine learning
- Model evaluation

## Examples Included

1. **text_preprocessing.py**: Text cleaning and preparation
2. **text_classification.py**: Classification with machine learning models

## Key Concepts

### Text Preprocessing
- Tokenization
- Lowercasing
- Removing punctuation and special characters
- Stop word removal
- Stemming and lemmatization

### Feature Extraction
- Bag of Words
- TF-IDF (Term Frequency-Inverse Document Frequency)
- N-grams

### Classification
- Naive Bayes
- Support Vector Machines
- Model evaluation metrics

## Running the Examples

```bash
cd 04_text_classification
python text_preprocessing.py
python text_classification.py
```

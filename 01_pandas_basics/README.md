# Pandas Basics

This module covers the fundamentals of the Pandas library, which is essential for data manipulation and analysis in Python.

## Learning Objectives

- Understand Pandas data structures (Series and DataFrame)
- Learn to create and manipulate DataFrames
- Master data selection and indexing techniques
- Perform basic data operations and transformations
- Use grouping and aggregation functions

## Examples Included

1. **pandas_basics.py**: Introduction to Series and DataFrames
2. **data_manipulation.py**: Common data manipulation operations
3. **grouping_aggregation.py**: Group operations and aggregations

## Key Concepts

### Series
A one-dimensional labeled array capable of holding any data type.

### DataFrame
A two-dimensional labeled data structure with columns of potentially different types.

## Running the Examples

```bash
cd 01_pandas_basics
python pandas_basics.py
python data_manipulation.py
python grouping_aggregation.py
```

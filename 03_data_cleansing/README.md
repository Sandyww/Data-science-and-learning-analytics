# Data Cleansing

This module covers essential data cleaning techniques for preparing data for analysis.

## Learning Objectives

- Identify and handle missing values
- Detect and treat outliers
- Normalize and transform data
- Remove duplicates
- Handle inconsistent data

## Examples Included

1. **missing_values.py**: Techniques for handling missing data
2. **outliers.py**: Outlier detection and treatment methods

## Key Concepts

### Missing Values
- Detection methods
- Imputation strategies
- Forward/backward filling
- Dropping vs. filling

### Outliers
- Statistical methods (IQR, Z-score)
- Visualization techniques
- Treatment strategies
- Impact on analysis

## Running the Examples

```bash
cd 03_data_cleansing
python missing_values.py
python outliers.py
```

# Sentiment Classification with Logistic Regression

This module focuses on sentiment analysis using feature engineering and logistic regression.

## Learning Objectives

- Understand sentiment analysis
- Perform feature engineering for text data
- Build sentiment classifiers using logistic regression
- Evaluate and optimize models
- Handle imbalanced datasets

## Examples Included

1. **feature_engineering.py**: Advanced feature engineering techniques
2. **sentiment_classification.py**: Sentiment analysis with logistic regression

## Key Concepts

### Feature Engineering
- Text features (TF-IDF, n-grams)
- Custom features (length, punctuation, etc.)
- Feature scaling and normalization
- Feature selection

### Logistic Regression
- Binary classification
- Regularization (L1, L2)
- Hyperparameter tuning
- Model interpretation

### Model Evaluation
- Accuracy, Precision, Recall, F1-score
- ROC curves and AUC
- Cross-validation
- Confusion matrices

## Running the Examples

```bash
cd 05_sentiment_classification
python feature_engineering.py
python sentiment_classification.py
```

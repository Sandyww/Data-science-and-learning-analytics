# Parsing Data (CSV and JSON)

This module covers techniques for reading and writing data in different formats, specifically CSV and JSON files.

## Learning Objectives

- Read and write CSV files
- Parse JSON data structures
- Handle different file encodings
- Work with nested JSON data
- Export data to various formats

## Examples Included

1. **csv_operations.py**: CSV file reading and writing
2. **json_operations.py**: JSON parsing and manipulation

## Key Concepts

### CSV (Comma-Separated Values)
- Simple tabular data format
- Easy to read and write
- Widely supported

### JSON (JavaScript Object Notation)
- Hierarchical data structure
- Flexible and human-readable
- Common in web APIs

## Running the Examples

```bash
cd 02_parsing_data
python csv_operations.py
python json_operations.py
```
